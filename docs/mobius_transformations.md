# Möbius Transformations and Hyperbolic Isometries

## What is a Möbius Transformation?

A Möbius transformation is a rational map of the Riemann sphere C ∪ {∞}:

```
f(z) = (az + b) / (cz + d)
```

where a, b, c, d ∈ ℂ and ad - bc ≠ 0. These are also called **fractional linear transformations**
or **bilinear transformations**.

The group of all Möbius transformations is PSL(2,ℂ) = SL(2,ℂ)/{±I}.

## Classification

A Möbius transformation f ≠ identity is classified by its trace:
```
trace = a + d   (sum of diagonal entries of the matrix [a,b;c,d])
```

| |trace|² | Type | Behavior | Fixed Points |
|-----------|------|----------|-------------|
| < 4 | **Elliptic** | Rotation-like | 2 fixed points (one in disk, one outside) |
| = 4 | **Parabolic** | Translation-like | 1 fixed point (on boundary) |
| > 4 (real) | **Hyperbolic** | Translation | 2 fixed points (both on boundary) |
| > 4 (complex) | **Loxodromic** | Spiral | 2 fixed points (both on boundary) |

## Isometries of the Poincaré Disk

The isometry group of the Poincaré disk is PSU(1,1), the group of Möbius transformations
that preserve the unit disk:

```
f(z) = e^(iθ) · (z - a) / (1 - ā·z)
```

where |a| < 1 and θ ∈ [0, 2π).

### Rotation

Setting a = 0: `f(z) = e^(iθ) z` — pure rotation about the origin.

```glsl
vec2 diskRotate(vec2 z, float theta) {
    float c = cos(theta), s = sin(theta);
    return vec2(c*z.x - s*z.y, s*z.x + c*z.y);
}
```

### Translation

Setting θ = 0: `f(z) = (z - a) / (1 - ā·z)` — moves a to 0.

```glsl
vec2 diskTranslate(vec2 z, vec2 a) {
    // f(z) = (z - a) / (1 - conj(a)·z)
    // In real 2D: complex multiplication via (u,v)·(x,y) = (ux-vy, uy+vx)
    vec2 num   = z - a;
    vec2 denom = vec2(1.0, 0.0) - vec2(a.x*z.x+a.y*z.y, a.y*z.x-a.x*z.y);
    float d2   = dot(denom, denom);
    return vec2(dot(num,denom), num.y*denom.x - num.x*denom.y) / d2;
}
```

### Rotation About a Point

To rotate by θ about the point p:
1. Translate p to origin: `z' = (z-p)/(1-p̄z)`
2. Rotate: `z'' = e^(iθ) z'`
3. Translate back: `z''' = (z''+p)/(1+p̄z'')`

```glsl
vec2 diskRotateAbout(vec2 z, vec2 p, float theta) {
    vec2 w  = diskTranslate(z, p);          // p → 0
    vec2 wr = diskRotate(w, theta);          // rotate about 0
    return diskTranslate(wr, vec2(-p.x, -p.y));  // 0 → p (inverse translation)
}
```

## The Upper Half-Plane Model

The upper half-plane ℍ = {z ∈ ℂ : Im(z) > 0} is another model for the hyperbolic plane.

The **Cayley map** `φ: ℍ → D` sends the upper half-plane to the Poincaré disk:
```
φ(z) = (z - i) / (z + i)
```

The inverse maps the disk to the upper half-plane:
```
φ⁻¹(w) = i(1 + w) / (1 - w)
```

The isometry group of ℍ is PSL(2,ℝ): transformations `f(z) = (az+b)/(cz+d)` with
a,b,c,d ∈ ℝ and ad - bc = 1.

In the upper half-plane:
- Geodesics are vertical lines and semicircles with center on the real axis
- The metric is: `ds² = (dx² + dy²) / y²`
- Isometries: `z → (az+b)/(cz+d)` with a,b,c,d ∈ ℝ, ad-bc = 1

## Fuchsian Groups

A **Fuchsian group** is a discrete subgroup of PSL(2,ℝ) (or equivalently PSU(1,1)).
It acts discontinuously on the hyperbolic plane.

Examples:
- Triangle groups (2,p,q): generated by three reflections in a triangle
- Fundamental groups of hyperbolic surfaces
- Arithmetic Fuchsian groups (using algebraic number fields)

A Fuchsian group has a **fundamental domain**: a connected region that contains exactly
one point from each orbit. The {7,3} tiling's fundamental triangle is a fundamental domain
for the (2,3,7) triangle group.

## Kleinian Groups

A **Kleinian group** is a discrete subgroup of PSL(2,ℂ). It acts on:
- The Riemann sphere ℂ ∪ {∞} (the conformal boundary of ℍ³)
- Hyperbolic 3-space ℍ³

The **limit set** Λ(Γ) ⊂ S² is the smallest closed Γ-invariant subset.
The **domain of discontinuity** Ω(Γ) = S² \ Λ(Γ) is where the group acts discontinuously.

### Classification of Kleinian Groups
- **Fuchsian**: Λ is a circle, group acts on a Poincaré disk
- **Quasifuchsian**: Λ is a Jordan curve (quasicircle), group acts on two disk-like domains
- **Totally degenerate**: Λ = S² (no domain of discontinuity!)
- **Kleinian**: general case, Λ can be a fractal of any Hausdorff dimension 0 ≤ d ≤ 2

## The Cayley Graph of a Möbius Group

The Cayley graph of a Fuchsian group Γ with generating set S:
- Vertices: elements of Γ
- Directed edges: (g, gs) for each s ∈ S

For the (2,3,7) rotation subgroup with generators S (order 7) and T (order 3):
- Each vertex has degree 7 + 3 = 10 (directed)
- The graph is an infinite tree-like structure embedded in the hyperbolic disk
- The vertex corresponding to the identity is at the origin

## Quasiconformal Maps and Teichmüller Theory

A **quasiconformal map** f: ℂ → ℂ satisfies the Beltrami equation:
```
∂f/∂z̄ = μ(z) · ∂f/∂z
```

where μ is the **Beltrami coefficient** with ||μ||∞ < 1.

Teichmüller space T(S) parametrizes all quasiconformal deformations of a surface S.
For a Fuchsian group Γ, the quasifuchsian space QF(S) ⊂ T(S) × T(S̄) is an open
neighborhood of the Fuchsian locus (the "straight" Fuchsian representations).

As we move through Teichmüller space:
- The tiling remains topologically equivalent
- But the geometry deforms
- The limit set evolves from a circle to a fractal Jordan curve
